# 🚖 PM Cabz - Smart Ride Booking Platform

## 📌 About the Project

**PM Cabz** ek modern aur efficient ride booking web application hai jo users ko fast, reliable aur seamless travel experience provide karta hai. Is platform ka main focus hai user-friendly interface ke saath smooth ride booking process dena, jahan user easily pickup aur destination select karke turant ride book kar sake.

## 🚀 Key Technologies

Is project ko powerful **MERN Stack** par develop kiya gaya hai. Frontend me **React.js** aur **Tailwind CSS** ka use karke clean aur responsive UI banaya gaya hai. Backend me **Node.js** aur **Express.js** application ko efficiently handle karte hain, jabki **MongoDB** data storage ke liye use hota hai.

## 📡 Real-Time Functionality

**Socket.io** ke through real-time communication implement kiya gaya hai, jisse live ride updates aur driver tracking possible hota hai. Isse user ko har moment par accurate ride status milta hai.

## 🗺️ Location & Tracking

Location-based services ke liye **Leaflet** integrate kiya gaya hai, jo interactive maps aur accurate route tracking provide karta hai. User apni ride ko real-time me track kar sakta hai.

## 🎯 Project Goal

PM Cabz ka goal hai ek scalable, fast aur reliable cab booking system banana jahan user bina kisi hassle ke ride book kare, vehicle choose kare aur apni journey ko live monitor kare.

## ✨ User Experience

Simple design, smooth navigation aur fast performance ke saath PM Cabz ek premium ride booking experience deliver karta hai jo modern users ki needs ko perfectly match karta hai.
